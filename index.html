🌡️ Sensors
Real sources. Real data.

ZXC.AIDA uses genuine Windows and vendor-backed sources for all telemetry data.

Source Type
Windows performance counters & runtime APIs Built-in
Windows ACPI thermal zones Built-in
Windows storage temperature properties Built-in
NVIDIA nvidia-smi Vendor
LibreHardwareMonitor / OpenHardwareMonitor WMI Optional

Some temperatures, fan speeds, voltages, and board-level sensors depend on hardware support, drivers, vendor tooling, or optional external monitor namespaces. DIMM temperature is only shown when a real source exposes it.
